# kanban
Simple kanban concept and side-project
## Structure
* Kanban board:
    * Board name
    * Department
    * Phases
        * CardId
    * Color?
* Kanban cards
    * BoardId
    * Create date
    * Due date
    * Priority
    * Topic
    * Description
    * Files
    * Comments
    * Owner
    * Members?
    * Phase
* Users
    * Name
    * Portrait image
    * Email
    * Departments
